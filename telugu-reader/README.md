# Telugu Reader (తెలుగు చదువరి)

A beautiful, modern, and comfortable text-to-speech reading web application designed for Telugu text. It splits text into Telugu and English chunks and utilizes speech synthesis to read them seamlessly.

## Features
- **Speech Playback controls**: Play, pause, and stop.
- **Speed & Pitch adjustment**: Dynamic speed control with session tracking.
- **Natural Voice Default**: By default, English words are read using the same selected Telugu speaker's voice to maintain a single natural persona.
- **Word Highlighting**: Visually tracks reading progress in real-time.

## Tech Stack
- React 19
- Vite
- HTML5 / Vanilla CSS (Modern glassmorphism UI)
- Web Speech API (SpeechSynthesis)

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd telugu-reader
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
